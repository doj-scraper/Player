# React + TypeScript + Vite

This is a modern web application built with React and TypeScript, using Vite as the build tool. It includes Tailwind CSS for styling and is configured for deployment on Vercel.

## Features

- React 18+
- TypeScript
- Vite
- Tailwind CSS
- ESLint and Prettier pre-configured
- Optimized for fast development and production builds

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview the production build:
   ```bash
   npm run preview
   ```

## Deployment

This project is configured for deployment on Vercel. Simply push your code to a Git repository connected to Vercel, or use the Vercel CLI to deploy manually.
